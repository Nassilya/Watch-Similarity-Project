# ⌚ Watch Similarity Search - Projet Big Data

Ce projet implémente un système de **recherche d'images par le contenu (CBIR)** appliqué à un dataset de montres. 
L'objectif est de fournir une image de montre et de retrouver les modèles les plus similaires dans notre base de données.

## Prérequis

Pour faire tourner ce projet, tu as besoin de :
